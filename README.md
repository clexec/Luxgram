# LuxGram

Модифицированный Telegram клиент для iOS на базе официального исходного кода.

## Возможности

- Ghost Mode — скрытие онлайна
- Double Bottom — скрытый аккаунт за паролем
- Fake Profile — изменение отображаемого профиля
- Удалённые сообщения — просмотр удалённых сообщений
- Fake Location — подмена геолокации
- Voice Morpher — изменение голоса
- Плагины — поддержка кастомных твиков
- Локальный Premium — разблокировка функций без подписки

## Сборка

```
./scripts/buildprod.sh
```

Требуется macOS, Xcode 16, Bazel.
